# Discount Price Export System

## Overview

This Python program:

* Stores product prices in a dictionary
* Takes a discount percentage as input
* Calculates discounted prices
* Writes results to a file
* Displays formatted output

---

## Features

* User input for discount percentage
* Calculates discounted price for each product
* Writes output to a file
* Displays:

  * Product name
  * Original price
  * Discounted price
* Summary:

  * Total items
  * Average discounted price

---

## Technologies Used

* Python (File Handling, Dictionary, Loops)

---

## File Structure

discount_price.txt   # Output file

README.md            # Documentation

## How It Works

1. A dictionary stores product prices
2. User enters discount percentage
3. Discount is applied to each product
4. Data is written to a file
5. Summary is added at the end

1. Open tthe ipynb folder
